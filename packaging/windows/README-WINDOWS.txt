Aegis Vault for Windows

What is included
- Aegis Vault.exe: the desktop app
- README-project.md: the project overview
- PACKAGE-INFO.txt: package version and build details
- launch-aegis-vault.bat: simple launcher

How to use
1. Extract the zip to a normal folder such as Desktop or Documents.
2. Double-click "Aegis Vault.exe" or "launch-aegis-vault.bat".
3. Create a vault folder in a location you control.
4. Keep regular backups of the vault folder.

Notes
- This first Windows package is a portable release, not an installer.
- Microsoft Defender may scan the app on first launch because it is a new unsigned executable.
- Do not place the vault inside a temporary folder.
- Keep the password safe. The app cannot recover it for you.

Recommended before sharing the build
- Start the app once on a clean Windows machine.
- Create a vault, import a file, export it, and verify the content.
- Run a health check and confirm the report looks normal.
